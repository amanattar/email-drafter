# Email Drafter Skill

A Claude skill for composing professional, effective business emails across any scenario.

## What It Does

Guides Claude to write emails that achieve a specific outcome — not just grammatically correct ones. Covers tone calibration, structure, scenario-specific templates, and strategic variants.

## When It Triggers

Any time you ask Claude to:
- Write or draft an email
- Improve or rewrite an existing email
- "Help me word this" or "how do I say this professionally"
- Respond to a difficult or sensitive message

## Scenarios Covered

Meeting requests, follow-ups, status updates, cold outreach, apologies, rejections, requests, escalations, introductions, salary negotiation, sensitive conversations, job applications, and client communication.
